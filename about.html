Lets talk about this blog

Why OSWE - AWAE? Because it is not easy to find a hard challenge nowadays, something that will push you above your limits, and on top of that developed by the famous Offsec training team. In my opinion there is no better resource for advanced web attacks and source code analysis. That certainly doesn't mean you won't suffer through trying harder. More information can be found: here.

This contains my path to AWAE (which is in progress still) from the point where I got OSCP. It has some custom beginner excercises I have done along with my notes, and extra mile excercises without any solution in offsec style. The main idea is to cover a lot of web stuff from many perspectives, which turned out to be really helpfull in my previous experience with offsec training - learn everything it might come in handy is my translation of the TRY HARDER moto and mantra. Despite the fact you will have to TRY HARDER, this is intended for everyone who is beggining with more serious web security, whether it's about some source code analysis, bug bounty public disclosures, ctfs. It is only the starting point and trial for something more serious that might have also AD, exploit dev and mobile security (as some would say, in the next two, maximally two and a half years).

Pre-requisites: try harder mantra, some basic owasp 10 knowledge and basic web. Take your time at each step to understand everything, you are, and you will be expected to do tons of independent research.
